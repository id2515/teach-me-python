Lesson

Goals

  • Understand variables as labeled boxes
  • Store text and numbers
  • Print values and sentences

How to explain it

A variable is a named place that stores information so your program can use it later.

Teacher move: Ask, “What do you think this code will do?” before running it.

Guided example

name = "Ava"
age = 15
print(name)
print(age)
print("My name is", name)

Try it

Change the values and predict the output before running.

Quick quiz
