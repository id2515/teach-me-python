25 short practice problems

Do 3 to 5 at a time. The goal is small wins.

  1. Create two variables, name and age, then print them.
  2. Ask the user for their favorite food and print a sentence with it.
  3. Ask for two numbers and print their sum.
  4. Ask for a number and print its double.
  5. Ask for age and print what it will be next year.
  6. Write an if statement that prints 'hot' if temperature > 80.
  7. Write an if/else program that checks whether a number is positive.
  8. Ask for a grade and print pass or fail.
  9. Use a for loop to print numbers 1 through 5.
  10. Use a while loop to print Hello three times.
  11. Make a list of three sports and print the second one.
  12. Add a new item to a list with append().
  13. Loop through a list of foods and print each one.
  14. Write a function called greet() that prints Hello.
  15. Write a function greet_name(name) that prints Hello and the name.
  16. Write a function square(n) that returns n*n.
  17. Make a calculator function add(a,b).
  18. Create a loop that keeps asking for a password until the user gets it right.
  19. Store inventory items in a list and print them.
  20. Build a tiny quiz that asks one question and keeps score.
  21. Make a Dog class with a bark() method.
  22. Create two Dog objects with different names.
  23. Make a Player class with health and an attack() method.
  24. Use random.randint(1,6) to simulate a dice roll.
  25. Build a mini menu program: 1=Play, 2=Help, 3=Quit.
