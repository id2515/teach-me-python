Full lesson plan: Rock Paper Scissors

Concept goals

  • Use input()
  • Use if / elif / else
  • Import random

Starter code

import random

choices = ["rock", "paper", "scissors"]
computer = random.choice(choices)
player = input("Choose rock, paper, or scissors: ").lower()

print("Computer chose:", computer)

Teaching sequence

  1. Ask the student how the computer could make a random choice.
  2. Type the starter code together.
  3. Have the student add rules for win, lose, and tie.
  4. Add a replay loop as a bonus challenge.
One possible solution 
import random

choices = ["rock", "paper", "scissors"]
computer = random.choice(choices)
player = input("Choose rock, paper, or scissors: ").lower()

if player == computer:
    print("Tie!")
elif (player == "rock" and computer == "scissors") or \
     (player == "paper" and computer == "rock") or \
     (player == "scissors" and computer == "paper"):
    print("You win!")
else:
    print("You lose!")
