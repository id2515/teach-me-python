Lesson

Goals

  • Write reusable code blocks
  • Use parameters
  • Return values

How to explain it

A function is a mini-program inside your program. You write it once and use it many times.

Teacher move: Ask, “What do you think this code will do?” before running it.

Guided example

def greet(name):
    print("Hello", name)

def add(a, b):
    return a + b

greet("Maya")
print(add(3, 4))

Try it

Write a function that returns the square of a number.

Quick quiz

Why use functions?

To reuse code and organize it.

What is a parameter?

An input the function receives.

What does return do?

It sends a value back from the function.
