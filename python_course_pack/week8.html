Lesson

Goals

  • Combine everything learned
  • Plan, build, test, improve
  • Present the project

How to explain it

This week is about creating something real. The student can make a text adventure or a mini RPG.

Teacher move: Ask, “What do you think this code will do?” before running it.

Guided example

inventory = []
health = 10
choice = input("Go left or right? ")
if choice == "left":
    inventory.append("key")
    print("You found a key!")

Try it

Sketch the story first on paper before coding.

Quick quiz

What should a good final project include?

A beginning, choices, logic, and a clear ending.

Why test every path?

So choices do not break the game.

What makes code easier to understand?

Comments, clear names, and functions.

Small assignment

Build a text adventure or mini RPG using variables, input, decisions, a loop, a list, and functions.
