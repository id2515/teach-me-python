Lesson

Goals

  • Make decisions in code
  • Use if, elif, else
  • Compare values

How to explain it

An if statement is a rule: if something is true, do this. If not, try another path.

Teacher move: Ask, “What do you think this code will do?” before running it.

Guided example

score = 88
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("Keep practicing")

Try it

Ask for temperature. If it is above 80, print Hot. Otherwise print Nice weather.

Quick quiz

What does if do?

It runs code only when a condition is true.

What is else for?

It runs when the earlier conditions are false.

What is the difference between = and == ?

= assigns a value, == compares values.

Small assignment

Build a grade checker using input and if / elif / else.
