Lesson

Goals

  • Install Python and VS Code
  • Create and run a .py file
  • Read simple errors calmly

How to explain it

This week is about confidence, not theory. The student only needs to learn how to type code, run it, and notice what changes.

Teacher move: Ask, “What do you think this code will do?” before running it.

Guided example

print("Hello world")
print("I am learning Python")

Try it

Change the text in each print statement, save, and run again.

Quick quiz

What does print() do?

It shows output on the screen.

What command usually runs a Python file?

python filename.py

If you see an error, what should you do first?

Read the error message and check the line number.

Small assignment

Create welcome.py that prints a welcome line, your name, and one fun fact.
